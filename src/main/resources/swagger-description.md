Сервис сравнения геокодеров от **Yandex** и **DaData**: вычисляет расстояние по координатам одного адреса в метрах. 
Принимает текстовый адрес, параллельно геокодирует его двумя источниками, считает расхождение между точками и сохраняет результат в БД.

## Что отдаёт API
По каждому источнику возвращаются координаты и уровень точности геокодера приведенный к единому формату (`HOUSE`, `STREET`, `LOCALITY`),
расхождение между точками `distanceMeters` в метрах и флаг доверия `reliable`, который истинен,
когда оба источника определили адрес до дома и сошлись в пределах порога расстояния.

Качество результата (`precision`, `reliable`) - это метаданные в теле ответа, а не код статуса: 
street-level или большое расхождение всё равно вернут `201`, решение о пригодности остаётся за потребителем.

## Формат ответов
- Успешный ответ - чистый объект результата без обёртки. 
- Ошибки возвращаются стандартным `ProblemDetail` с полями `status`, `title`, `detail`;
при недоступности геокодера добавляется `provider`, при ошибке валидации - `errors`.

## Коды ответов
API использует стандартные HTTP-коды:
* `201` - сравнение создано,
* `200` - сохранённое сравнение получено по идентификатору,
* `400` - невалидный адрес, 
* `404` - сравнение по идентификатору не найдено, 
* `422` - хотя бы один источник не нашёл адрес, 
* `502` - геокодер недоступен.
